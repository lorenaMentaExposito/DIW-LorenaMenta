Lista de Definiciones

  • HTML: Lenguaje de marcado para estructurar contenido web.
  • CSS: Lenguaje para dar estilo a los documentos HTML.
  • JavaScript: Lenguaje de programación para crear interactividad en la web.
  • DOM: Modelo de objetos que representa la estructura de un documento HTML.
  • API: Conjunto de reglas para la comunicación entre aplicaciones.
